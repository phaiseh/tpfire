# How to Cite

If you wish to cite content from this repository in research, news reports, or publications, please refer to the format below.

## General Citation

When citing **original content** curated in this repository (e.g., timelines, analysis reports, architecture), please use the following format:

```text
Source: 11.26 Tai Po Wang Fuk Court Fire Documentary (Hong Kong Fire Documentary)
URL: https://github.com/Hong-Kong-Emergency-Coordination-Hub/Hong-Kong-Fire-Documentary
License: CC BY-SA 4.0
```

## Citing Specific Documents

If citing **government documents, news reports, or external reports** within this repository, please attribute both the original source and this repository as the archive source:

### Example

> According to "Engineering Analysis on Improving Safety and Reliability of Bamboo Scaffolding" published by OSHC (Archived in Hong Kong Fire Documentary)...
